# Task- 25 - SELECT Queries

## Objective
To learn how to retrieve data from a database table using SQL SELECT statements.

## Program

```sql
CREATE TABLE students (
    id INT,
    name VARCHAR(50),
    marks INT
);

INSERT INTO students VALUES
(1,'Arun',85),
(2,'Bala',90),
(3,'Charan',75);

SELECT * FROM students;
SELECT name FROM students;
SELECT marks FROM students;
```

## Explanation

- SELECT * displays all columns.
- SELECT name displays only student names.
- SELECT marks displays only marks.

## Output
Retrieved complete and specific data from the students table.

## Outcome
Learned how to retrieve records from a database using SELECT queries.
